## Thread safe queue

Light thread safe queue implementation

### Installation

	1. Clone the repo 
	   ```sh
	   https://github.com/tabrisovstepan/thread_safe_queue.git
	   ```
	2. Install
	   ```sh
	   cd thread_safe_queue/
	   mkdir .build && cd .build
	   cmake ../
	   sudo make install
	   ```
